## Share Your Bootnode

Want to help the network? Run a node and share your bootnode!

**Where to share:**
- BitcoinTalk thread
- Crypto forums
- Your own channels

**Your connection string format:**
```
/ip4/YOUR_IP/tcp/30333/p2p/YOUR_PEER_ID
```

**How to get your Peer ID:**
```bash
curl -s -H 'Content-Type: application/json' -d '{"id":1,"jsonrpc":"2.0","method":"system_localPeerId"}' http://127.0.0.1:9944
```
